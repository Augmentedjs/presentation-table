new AutomaticTable()

Description

AutomaticTable

Creates a table automatically via a schema for defintion and a uri/json for data

Examples
const at = new AutomaticTable({
			    schema: schema,
			    el: "#autoTable",
			    crossOrigin: false,
			    sortable: true,
			    lineNumbers: true,
			    editable: true,
			    uri: "/example/data/table.json",
			    messagePosition: "top", // or bottom
			    theme: "material",
			    linkable: true,
			    links: {
			       wholeRow: true,
			       column: "",
			       link: "rowLink"
			    },
			    fetchOnStart: false
			});
